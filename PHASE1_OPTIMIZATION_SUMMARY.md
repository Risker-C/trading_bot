## 🎯 预期收益计算

### 优化前收益模型
```
单笔收益 = 3% 仓位 × 10x 杠杆 × 3% 止盈 = 0.9%
日交易次数 = 3-4次
日收益率 = 0.9% × 3.5次 = 3.15%（理想情况）
实际收益 ≈ 0.5-1%（考虑亏损）
```

### 优化后收益模型
```
单笔收益 = 5% 仓位 × 10x 杠杆 × 5% 止盈 = 2.5%
日交易次数 = 5-7次（信号阈值降低）
日收益率 = 2.5% × 6次 = 15%（理想情况）
实际收益 ≈ 2-3%（考虑亏损）
套利收益 = +0.5%
总收益 ≈ 2.5-3.5%
```

**收益提升**: 2-3倍
